# Sistema-de-adoção-de-animais
Site que conecta ongs, veterinários, usuários para adotar e doar animais entre outros serviços.
INSTRUÇÕES DE USO:
Ao executar o arquivo index.html, você será direcionado para a tela inicial do sistema, onde encontrará o menu no canto direito, um filtro de busca de ONGs por cidades, o slogan acompanhado de opções do menu e, logo abaixo, o rodapé com botões funcionais e o contato fictício da plataforma.
O botão “Quero Adotar” leva ao cadastro de adotantes, onde o usuário deve preencher os campos obrigatórios para entrar na fila de adoção.
O botão “Quero Doar um Animal” abre o formulário de cadastro do animal, também com campos obrigatórios.
Na página Doações, é possível contribuir com alimentos, medicamentos, dinheiro e outros itens.
O botão “Fazer Denúncia” direciona para um formulário onde o usuário descreve a denúncia, que é encaminhada para uma ONG responsável.
O botão “ONGs” exibe as organizações disponíveis, permitindo visualizar animais para adoção, entrar em contato e realizar doações.
A página Contato oferece um formulário para comunicação direta com os administradores do sistema, com campos obrigatórios.
O botão Sobre apresenta a história de criação do sistema.
O botão Emergência abre um formulário para preenchimento dos dados obrigatórios e, ao acionar a rede de emergência, o sistema busca veterinários ou ONGs disponíveis para atendimento imediato.
A tela de Login e Cadastro permite criar uma conta e acessar o sistema, com opções de login via Facebook e Google.
